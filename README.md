本项目仅界面部分是开源的，下载的二进制文件也是限期试用，试用结束仍可正常使用，但会限制一些功能。具体如下：

||试用到期|试用/已购|
| - | - | - |
|打开文件的大小限制| < 100M | 无限制
|打开文件| 不可用| 无限制
|导出过滤结果|不可用|无限制
|打开外部程序（如adb）|不可用|无限制

如介意，请勿下载。

（深知纯开源、靠赞助是不可能一直做下去的。只要有人愿意付费，就说明值得继续做，才能坚持挤时间维护）

# 版权声明
